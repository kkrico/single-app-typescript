# Single APP Typescript

Exemplo de um monorepo contendo:

- Node (Express)
- React
- Logica e variaveis de ambiente compartilhadas

# Dependencias
  
  - node
  - yarn
  
# Build

Instale as dependencias
```bash
yarn install
```

Rode
```bash
yarn start:dev
```
